F1 Machine Learning Project

This project analyzes Formula 1 race data using Python, FastF1, pandas, and scikit-learn. The notebook loads race telemetry, processes lap data, and applies machine learning models to predict lap times and classify drivers.

Key Features

  • Loads and processes F1 telemetry and lap data for the 2023 season
  • Performs data cleaning and feature engineering
  • Visualizes correlations with heatmaps
  • Applies multiple linear regression and KNN to predict lap times
  • Uses logistic regression to classify if a lap was driven by Max Verstappen
