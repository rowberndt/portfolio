Time Tracker App

The Time Tracker App is a desktop application designed to help users monitor and analyze how they spend their time each day. With an intuitive interface, users can log time spent on various activities, visualize their daily and historical trends, and gain insights to improve productivity and balance.

Key Features

  • Customizable categories for tracking any activity
  • Pie chart visualization of daily time allocation
  • Line chart visualization of trends over time
  • Persistent data storage using CSV files
  • Modern, user-friendly PyQt6 interface
