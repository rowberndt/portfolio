Time Tracker App

This is a desktop application for tracking time spent on various activities, built with Python and PyQt6. It allows users to record time by category, visualize their time with pie and line charts, and save/load data from CSV files.

Key Features

  • Track time by customizable categories
  • Pie chart visualization of daily time allocation
  • Line chart visualization of category trends over days
  • CSV-based data persistence
  • Simple, interactive PyQt6 GUI
